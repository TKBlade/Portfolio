ôdecasa - Sistema de arrendamentos

Desenvolvimento de aplicativo similar ao Airbnb, seguindo a metodologia Scrum com quatro sprints de 7 dias e gráfico Burn-down para medir a entrega. O Back-end utiliza Java e Spring Boot com MySQL e exporta uma API. No Front-end, é utilizado principalmente React para consumo da API e experiência suave. O deploy foi realizado em Docker através da AWS, com testes automatizados e gitlab runner para CI/CD, Terraform para infraestrutura e AWS Codedeploy
